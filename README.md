# Customer Retention Intelligence System

## Overview

Customer Retention Intelligence System is an end-to-end Machine Learning project designed to help businesses identify customers who are likely to churn and proactively take retention actions.

The system predicts customer churn probability, classifies customers into risk tiers, segments customers based on business value and engagement, and generates personalized retention recommendations.

---

## Features

* Customer Churn Prediction using Machine Learning
* Risk Tier Classification (Low, Medium, High)
* Customer Segmentation
* Retention Priority Scoring
* Rule-Based Retention Recommendation Engine
* Interactive Streamlit Dashboard
* Real-Time Customer Analysis

---
