# Customer Retention Intelligence System 🚀

An AI-powered customer churn prediction and retention recommendation system built using Machine Learning and Streamlit.

## 📌 Project Overview

This project analyzes customer behavior and predicts customers who are likely to churn. 
It provides customer risk analysis and helps businesses take data-driven retention decisions.

## 🎯 Problem Statement

Customer churn directly impacts business revenue. 
The goal of this project is to identify high-risk customers before they leave and suggest retention strategies.

## ✨ Features

- Customer churn prediction
- Customer risk classification
- Customer analytics dashboard
- Interactive visualizations
- Customer behavior analysis
- Retention insights
- Machine Learning based prediction
- Streamlit web application

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Plotly
- Streamlit
- Joblib

## 📊 Dashboard

The application provides:
